An app created to find recipes for Pokemon Sword&Shield Isle of Armor expansion to ease using of Cram-o-Matic.

Cram-o-matic description and recipes, serebii.net: https://serebii.net/swordshield/cram-o-matic.shtml
Bulbapedia description, recipes, etc: https://bulbapedia.bulbagarden.net/wiki/Cram-o-matic

Items.py served as a GUI to help fill database correctly or edit it
Recipes.py is a main part of an app that finds recipes in database.
